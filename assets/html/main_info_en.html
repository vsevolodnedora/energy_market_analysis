This project uses advanced MLOps pipelines to produce accurate week-ahead forecasts for renewable energy generation, electricity load, and prices. Combining robust data engineering, modern machine learning, and automated workflows, it optimizes operations, enhances trading strategies, and promotes sustainable energy integration.

Source code: GitHub Repository

Value Proposition

  • Optimize Operations: Efficient maintenance schedules minimize disruptions and costs.
  • Enhance Trading: Improved bidding strategies and reduced financial risk.
  • Support Grid Stability: Effective supply-demand balance for renewable integration.
  • Drive Sustainability: Actionable insights for decarbonization and long-term planning.

Core Features & Workflow

  • Daily Data Collection: Automatic retrieval from APIs ( ENTSO-E, SMARD, openmeteo, EPEXSPOT and EEX ).
  • Preprocessing: Tailored cleaning and feature engineering for wind, solar, and load data.
  • Forecasting Models: Prophet, XGBoost, and Elastic Net, with ensemble methods for higher accuracy.
  • Hyperparameter Optimization: Automated tuning (Optuna) to minimize forecast error.
  • Pipeline Automation: GitHub Actions for continuous data updates and deployment.

Technical Highlights

  • Multi-Step Forecasting: Hourly forecasts for an entire week.
  • Advanced Feature Engineering: Weather-driven inputs (wind power density, solar radiation, etc.).
  • Interactive Visualizations: Charts built with ApexCharts.
  • Extendability: Explainable models (SHAP) and customizable pipelines.
